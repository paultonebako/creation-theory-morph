# Changelog — Creation Theory Morph

All notable changes to Creation Theory Morph will be documented here.

## [1.0.0] — 2026-04-27

### Initial Release

- 3D OpenGL viewport with rotate, pan, and zoom
- Wireframe, Solid, and Solid+Wireframe display modes
- STL, OBJ, PLY, OFF, GLB/GLTF import via trimesh
- STL, OBJ, PLY export
- Batch folder import
- Object browser dock with per-object visibility toggle
- Mesh tools: flip normals, merge vertices, mesh statistics
- Built-in demo objects: cube, sphere, torus
- Dark theme UI
- Axis and grid display with toggle shortcuts
- Status bar with live viewing angle readout
- PyInstaller-compatible for standalone packaging
